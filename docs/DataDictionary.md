# Data Dictionary

## distillery

| Field | Definition |
|---|---|
| distillery_id | Permanent identifier for one physical distillery site. |
| official_name | Official or preferred current name. |
| region_id | Controlled reference to region table. |
| type | Malt, Grain, or Malt & Grain. |
| status | Operating, Silent, Mothballed, Under Construction, Planned, Closed. |
| founded_year | Four-digit year where verified. |
| owner_id | Current operating owner. |
| parent_company_id | Ultimate controlling parent company. |
