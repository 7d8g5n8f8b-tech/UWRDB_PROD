# Data Governance

## Rules

1. Unknown is better than wrong.
2. Every verified field requires evidence.
3. Every release must be reproducible.
4. A physical distillery appears once.
5. Brands and expressions are linked entities, not duplicate distilleries.

## Evidence strength

| Score | Meaning |
|---:|---|
| 5 | Official source plus corroboration |
| 4 | Official source |
| 3 | Reliable secondary source |
| 2 | Limited/conflicting evidence |
| 1 | Historic/uncertain |
