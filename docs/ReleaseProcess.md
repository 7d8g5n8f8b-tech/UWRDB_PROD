# Release Process

## Regional release

1. Curate records in `data/curated/`.
2. Add evidence rows.
3. Run build.
4. Review QA report.
5. Copy the release bundle into `releases/<Region>/<ReleaseName>/`.
6. Update `releases/RELEASE_INDEX.md`.

## Scotland-wide release

A Scotland release should only be produced after every regional release has zero QA blockers.
