# Curation Checklist

For every distillery:

- [ ] One physical site only
- [ ] Stable `AVD-*` ID
- [ ] Region normalized
- [ ] Type normalized
- [ ] Status normalized
- [ ] Owner normalized
- [ ] Parent company normalized
- [ ] Evidence for official name
- [ ] Evidence for owner
- [ ] Evidence for region/status
- [ ] QA blockers resolved

For Gold:

- [ ] Address verified
- [ ] Website verified
- [ ] Coordinates verified
- [ ] Technical fields verified or left blank
- [ ] Open research items documented
