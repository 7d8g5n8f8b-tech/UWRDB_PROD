# Campbeltown Gold Candidate v0.1

Generated: 2026-07-01

## Scope

This is the first showable UWRDB/Aqua Vitae regional release.

It covers the three operating Campbeltown whisky distilleries:

- Glen Scotia
- Springbank
- Glengyle

It also models the main associated official brands:

- Glen Scotia
- Springbank
- Longrow
- Hazelburn
- Kilkerran

## Data quality position

This release is intentionally conservative.

Identity, ownership, address, visitor/tour and brand relationships are strongly sourced from official pages wherever possible.

Technical production fields are included where supported, but some remain marked as Gold Candidate because they require owner-level confirmation before a final Gold release.
