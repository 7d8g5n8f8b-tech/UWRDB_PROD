# UWRDB – Universal Whisky Reference Database

## Project
Aqua Vitae – Magnum Opus

## Mission
Build an evidence-based, versioned reference database for whisky, beginning with Scotland.

## Principles
- One physical distillery = one canonical record.
- Every fact has evidence.
- Every change is versioned.
- Unknown is better than incorrect.
- Excel is an export, not the source of truth.
