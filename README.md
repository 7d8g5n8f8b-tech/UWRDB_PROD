# UWRDB – Universal Whisky Reference Database

**Project name:** Aqua Vitae – Magnum Opus  
**Initial domain:** Scotch whisky distilleries  
**Goal:** Build a curated, evidence-backed, versioned reference database for whisky.

## Core principle

Excel is an output, not the source of truth.

The source of truth lives in structured CSV files, SQL schema, evidence records, and reproducible build scripts.

## Current release

`v0.2.0 – Foundation + Build System`

This release adds:

- Stronger relational schema
- Seed lookup tables
- QA rules
- Python build/validation script
- SQLite export path
- CSV release structure
- Documentation skeleton
