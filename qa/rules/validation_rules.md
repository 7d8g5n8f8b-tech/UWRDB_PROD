# Validation Rules

## Required

- All primary keys are unique.
- All foreign keys resolve.
- Distillery type is controlled: Malt, Grain, Malt & Grain, Unknown.
- Status is controlled: Operating, Silent, Mothballed, Under Construction, Planned, Closed, Unknown.
- Confidence must be 1–5.
- Verified evidence requires a source_id.

## Recommended

- Website should use HTTPS.
- Coordinates should be present for Gold records.
- Operating distilleries should have at least one evidence record.
